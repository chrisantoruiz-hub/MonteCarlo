Determination of MCP and IC Efficiencies from Coincidence Spectra

1. Experimental Geometry

Particles enter the system with total number \(N_{\text{tot}}\) and traverse:

  1. Grid 1 with transmission \(\tau_{\text{grid1}}\)
  2. MCP1 wire planes with transmission \(\tau_{\text{wires1}}\)
  3. MCP2 wire planes with transmission \(\tau_{\text{wires2}}\)
  4. Grid 2 with transmission \(\tau_{\text{grid2}}\)

The MCP detector efficiencies are \(\eta_{\text{MCP1}},\quad \eta_{\text{MCP2}}\) and we define the combined MCP coincidence efficiency:

\[\eta_{\text{MCP}} = \eta_{\text{MCP1}}\eta_{\text{MCP2}}.\]

The ion chamber efficiency is \(\eta_{\text{IC}}\).

2. Counting Equations

MCP Coincidence Spectrum

\[N_{\text{MCP}} = N_{\text{tot}}\;\tau_{\text{grid1}}\;\tau_{\text{wires1}}\;\tau_{\text{wires2}}\;\eta_{\text{MCP}}\]

IC Spectrum

\[N_{\text{IC}} = N_{\text{tot}}\;\tau_{\text{grid1}}\;\tau_{\text{wires1}}\;\tau_{\text{wires2}}\;\tau_{\text{grid2}}\;\eta_{\text{IC}}\]

Overlap Spectrum (MCP ∩ IC)

\[N_{\text{MCP}\cap\text{IC}} = N_{\text{tot}}\;\tau_{\text{grid1}}\;\tau_{\text{wires1}}\;\tau_{\text{wires2}}\;\tau_{\text{grid2}}\;\eta_{\text{MCP}}\;\eta_{\text{IC}}\]

3. Extraction of Efficiencies

\[\boxed{\eta_{\text{MCP}} = \frac{N_{\text{MCP}\cap\text{IC}}}{N_{\text{IC}}}}\] \[\boxed{\eta_{\text{IC}} = \frac{N_{\text{MCP}\cap\text{IC}}}{N_{\text{MCP}}\,\tau_{\text{grid2}}}}\]

All upstream transmission factors and \(N_{\text{tot}}\) cancel.

4. Numerical Example

Assume:

\[N_{\text{tot}} = 100,\quad \tau_{\text{grid1}} = 0.95,\quad \tau_{\text{grid2}} = 0.90\] \[\tau_{\text{wires1}} = \tau_{\text{wires2}} = 0.98^3 = 0.941192\] \[\eta_{\text{MCP}} = 0.95,\quad \eta_{\text{IC}} = 0.95\]

Intermediate Transmission

\[\tau_{\text{up}} = 0.95 \times (0.941192)^2 = 0.84155\] \[\tau_{\text{full}} = 0.84155 \times 0.90 = 0.75740\]

Counts

\[N_{\text{MCP}} = 100 \times 0.84155 \times 0.95 = 79.95\] \[N_{\text{IC}} = 100 \times 0.75740 \times 0.95 = 71.95\] \[N_{\text{MCP}\cap\text{IC}} = 100 \times 0.75740 \times 0.95^2 = 68.38\]

Efficiencies are recovered:

\[\eta_{\text{MCP}} = \frac{68.38}{71.95} = 0.95\] \[\eta_{\text{IC}} = \frac{68.38}{79.95 \times 0.90} = 0.95\]

5. Poisson Counting Statistics

Assume each measured count follows Poisson statistics: \(\sigma_N = \sqrt{N}\). Thus,

\[\sigma_{N_{\text{MCP}}} = \sqrt{N_{\text{MCP}}},\quad \sigma_{N_{\text{IC}}} = \sqrt{N_{\text{IC}}},\quad \sigma_{N_{\text{MCP}\cap\text{IC}}} = \sqrt{N_{\text{MCP}\cap\text{IC}}}\]

Uncertainty on \(\eta_{\text{MCP}}\)

Since \(\eta_{\text{MCP}} = N_{\text{MCP}\cap\text{IC}} / N_{\text{IC}}\), error propagation gives

\[\left( \frac{\sigma_{\eta_{\text{MCP}}}}{\eta_{\text{MCP}}} \right)^2 = \frac{1}{N_{\text{MCP}\cap\text{IC}}} + \frac{1}{N_{\text{IC}}}.\]

Thus

\[\sigma_{\eta_{\text{MCP}}} = \eta_{\text{MCP}}\sqrt{\frac{1}{N_{\text{MCP}\cap\text{IC}}} + \frac{1}{N_{\text{IC}}}}.\]

Uncertainty on \(\eta_{\text{IC}}\)

Since \(\tau_{\text{grid2}}\) is constant,

\[\left( \frac{\sigma_{\eta_{\text{IC}}}}{\eta_{\text{IC}}} \right)^2 = \frac{1}{N_{\text{MCP}\cap\text{IC}}} + \frac{1}{N_{\text{MCP}}}.\]

Therefore,

\[\sigma_{\eta_{\text{IC}}} = \eta_{\text{IC}}\sqrt{\frac{1}{N_{\text{MCP}\cap\text{IC}}} + \frac{1}{N_{\text{MCP}}}}.\]

6. Numerical Statistical Uncertainties

Using the example counts \(N_{\text{MCP}} = 79.95\), \(N_{\text{IC}} = 71.95\), \(N_{\text{MCP}\cap\text{IC}} = 68.38\):

\[\sigma_{\eta_{\text{MCP}}} = 0.95\sqrt{\frac{1}{68.38} + \frac{1}{71.95}} = 0.16\] \[\sigma_{\eta_{\text{IC}}} = 0.95\sqrt{\frac{1}{68.38} + \frac{1}{79.95}} = 0.15\]

Conclusion

Detector efficiencies can be extracted purely from coincidence ratios. Under Poisson statistics, the fractional uncertainty is determined by the inverse square roots of the measured counts.
